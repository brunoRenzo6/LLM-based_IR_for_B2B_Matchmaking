# LLM-based Information Retrieval for B2B Matchmaking

Project developed at the Institute of Computing (IC) of the University of Campinas (UNICAMP) in partial fulfillment of the requirements for the degree of Master in Computer Science.

Supervisor: [Prof. Dr. Marcelo da Silva Reis](https://orcid.org/0000-0002-3754-9115)

In parternship w/: [MatchIT](https://site.matchit.com.br/)


Demo of a B2B Matchmaking application built using the LLM-based IR architecture proposed in our dissertation and article:

https://github.com/user-attachments/assets/dd303443-931a-4b6c-9530-92b1317c218f

(Note that the “summarizing snippets” process is not based on the project description, but only on the seller snippets. This way we avoid the summaries being biased towards the project descriptions and indeed present reliable summaries of the sellers.)

In this repo we present an AI refactored version of the raw code used during the research. The goal is to enhance the presentation of the architecture, since the raw (human written) code was, in fact, distributed in many .ipynb.
